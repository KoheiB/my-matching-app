# MyPartner
## Overview

あなたの人生を共に歩んでくれるパートナーを探すことができます。

## Features

- 認証機能
- 無限スクロール機能
- フィルタリング/ソートによる検索機能
- 登録ユーザーにいいねを送り、承認されたらマッチングする機能
- マッチングしたユーザーとメッセージのやり取りをする機能
- 未読メッセージ数を表示する機能
- プロフィール編集/写真アップロード機能

## 実装予定機能

- 通知機能（新着いいね、新着メッセージ）
- 無料/有料会員ステータス切換機能
## Technology

- JavaScript Framework: Nuxt.js
- Styling: Vuetify
- BaaS: Firebase
  - IDaaS: Firebase Authentication

  - : Cloud Functions
  - : Cloud Storage
  - Hosting: Firebase Hosting
